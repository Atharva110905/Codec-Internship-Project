# Codect Technologies Internship Projects

## Project 1 — Customer Churn Prediction
![Churn Dashboard](churn_dashboard.png)

**Goal:** Predict which customers are likely to stop using a service.  
**Tools:** Power BI | Models: Logistic Regression, Random Forest, XGBoost  
**Dataset:** Telecom customer data (7,043 records)  
**Key finding:** Month-to-month contract customers churn at 3x the rate of annual contract customers.

## Project 2 — Sales Forecasting
![Sales Dashboard](sales_dashboard.png)

**Goal:** Predict future sales based on historical data.  
**Tools:** Power BI | Models: ARIMA, Prophet, LSTM  
**Dataset:** Retail e-commerce (2024–2025)  
**Key finding:** Promotional months drive 38% higher revenue; Q4 peaks due to holiday season.
